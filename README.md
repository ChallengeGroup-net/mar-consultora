## 🚀 Uso local

No requiere instalación ni servidor. Simplemente:

```bash
# Clonar el repositorio
git clone https://github.com/ChallengeGroup-net/mar-consultora.git

# Abrir en el navegador
cd mar-consultora
open MAR_Consultora_updated.html
```

O arrastrá el archivo `.html` directamente al navegador.

---

## ✏️ Personalización

El sitio usa variables CSS centralizadas en `:root`. Para cambiar colores o tipografía, editá únicamente esa sección al inicio del `<style>`:

```css
:root {
  --navy: #001B2A;
  --gold: #B8955A;
  /* ... */
}
```
